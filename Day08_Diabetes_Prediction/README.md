# Diabetes Prediction

## Objective

Predict whether a person has diabetes or not using health-related features such as age, BMI, blood glucose level, HbA1c level, hypertension, heart disease history, and smoking habits.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

## Algorithms Used

* Linear Regression
* K-Nearest Neighbors (KNN)
* Decision Tree Classifier
* Random Forest Classifier
* AdaBoost Classifier

## Data Preprocessing

* Checked for missing values
* Removed duplicate records
* Encoded categorical variables using Label Encoding
* Prepared features and target variable for model training

## Model Performance

| Model             | Accuracy |
| ----------------- | -------- |
| Linear Regression | 93.75%   |
| KNN               | 95.15%   |
| Decision Tree     | 94.74%   |
| Random Forest     | 96.86%   |
| AdaBoost          | 97.06%   |

## Key Findings

* The dataset contained over 96,000 patient records.
* AdaBoost achieved the highest accuracy among all models.
* Blood glucose level, HbA1c level, BMI, and age played an important role in diabetes prediction.
* The models demonstrated strong performance in classifying diabetic and non-diabetic patients.

## Conclusion

AdaBoost achieved the best performance with an accuracy of 97.06%, demonstrating that machine learning can effectively assist in diabetes prediction and early healthcare decision-making.
