# Digit Recognition using Neural Networks

## Objective

Build a digit recognition model using the MNIST dataset to classify handwritten digits from 0 to 9 using a Feedforward Neural Network.

## Dataset

* MNIST Handwritten Digits Dataset
* 42,000 grayscale images
* Each image represented by 784 pixel values (28×28 pixels)

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* TensorFlow
* Keras
* Scikit-learn
* Jupyter Notebook

## Project Workflow

1. Loaded and explored the MNIST dataset.
2. Performed data preprocessing and normalization.
3. Split the dataset into training and testing sets.
4. Built a Feedforward Neural Network using TensorFlow and Keras.
5. Trained the model on handwritten digit images.
6. Evaluated model performance using accuracy, confusion matrix, and classification report.

## Model Performance

* Test Accuracy: **96.21%**

## Key Findings

* Successfully classified handwritten digits from 0 to 9.
* Achieved a test accuracy of 96.21%.
* Most digit classes achieved precision and recall values above 95%.
* The neural network effectively learned patterns from handwritten images.

## Conclusion

This project demonstrated the effectiveness of neural networks for image classification tasks. By training a Feedforward Neural Network on the MNIST dataset, the model achieved 96.21% accuracy and successfully recognized handwritten digits with high precision.
