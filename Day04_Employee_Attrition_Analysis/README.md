# Employee Attrition Analysis

## Objective
Analyze employee attrition data to identify factors affecting employee retention and provide recommendations for improving retention.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data Cleaning
- Attrition Analysis
- Department-wise Analysis
- Overtime Analysis
- Job Satisfaction Analysis
- Income Analysis
- Work-Life Balance Analysis

## Key Findings
- Employees working overtime were more likely to leave.
- Lower job satisfaction was associated with higher attrition.
- Lower-income employees showed higher turnover.
- Better work-life balance improved employee retention.

## Recommendations
- Reduce excessive overtime.
- Improve employee satisfaction programs.
- Review compensation structures.
- Promote a healthy work-life balance.

## Outcome
The analysis provided insights into employee retention challenges and suggested strategies to reduce attrition.
