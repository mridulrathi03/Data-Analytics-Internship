# Breast Cancer Classification

## Objective
Classify breast cancer tumors as malignant or benign using machine learning algorithms.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Algorithms Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree Classifier
- Random Forest Classifier

## Results
| Model | Accuracy |
|---------|---------|
| Logistic Regression | 95.61% |
| KNN | 95.61% |
| Decision Tree | 94.74% |
| Random Forest | 96.49% |

## Key Findings
- The dataset contained 569 breast cancer records.
- Benign tumors were more common than malignant tumors.
- Random Forest achieved the highest accuracy.
- The model correctly classified most tumor cases.

## Conclusion
Random Forest achieved the best performance with an accuracy of 96.49%, showing that machine learning can effectively classify breast cancer tumors.
