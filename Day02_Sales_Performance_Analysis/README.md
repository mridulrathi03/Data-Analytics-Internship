# Sales Performance Analysis

## Objective
Analyze sales performance across regions, categories, and time to identify business trends and top-performing products.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data Cleaning
- KPI Analysis
- Regional Performance Analysis
- Category Analysis
- Product Analysis
- Seasonality Analysis

## Key Findings
- The West region generated the highest sales.
- Technology was the top-performing category.
- Phones were the highest-selling sub-category.
- Sales peaked during November.

## Recommendations
- Focus marketing efforts on high-performing regions.
- Increase inventory for top-selling products.
- Launch seasonal campaigns during peak sales periods.

## Outcome
The analysis identified sales trends, top-performing products, and opportunities to improve business performance.
