# Sentiment Analysis on Movie Reviews

## Objective

Perform sentiment analysis on movie reviews to classify reviews as positive or negative using machine learning techniques.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn
* XGBoost
* Jupyter Notebook

## Algorithms Used

* Logistic Regression
* Naive Bayes Classifier
* K-Nearest Neighbors (KNN)
* Decision Tree Classifier
* Random Forest Classifier
* XGBoost Classifier

## Data Preprocessing

* Checked for missing values
* Removed duplicate reviews
* Converted sentiment labels into numerical values
* Applied TF-IDF Vectorization for text feature extraction

## Model Performance

| Model               | Accuracy |
| ------------------- | -------- |
| Logistic Regression | 88.52%   |
| Naive Bayes         | 85.03%   |
| KNN                 | 72.18%   |
| Decision Tree       | 71.51%   |
| Random Forest       | 84.04%   |
| XGBoost             | 85.23%   |

## Key Findings

* The dataset contained nearly 50,000 movie reviews.
* Logistic Regression achieved the highest accuracy among all models.
* Positive and negative reviews were almost equally distributed.
* TF-IDF vectorization successfully transformed textual reviews into machine-learning-friendly features.

## Conclusion

Logistic Regression achieved the best performance with an accuracy of 88.52%, demonstrating the effectiveness of machine learning and natural language processing techniques for sentiment analysis tasks.
